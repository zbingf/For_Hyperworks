# CheckElemAttachTie

### 软件版本
+ hypermesh 2017

### 功能
+ 根据Tie接触的单元,检查网格单元是否完全共节点, 仅显示未连接的网格


-----------------
### 控制参数
无控制参数

-----------------
### 操作
直接点击计算


-----------------
## 原理
+ 仅显示groups的单元(Tie-点对面),获取单元ID 1
+ 显示全部网格, 获取groups(Tie-面对面)对应单元ID 2
+ 拼接单元ID 1与ID 2, 根据"by Attached" 获取 ID 3
+ 隐藏ID 3单元
