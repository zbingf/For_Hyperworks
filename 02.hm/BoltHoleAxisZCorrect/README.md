# BoltHoleAxisZCorrect

### 软件版本
+ hypermesh 2017

### 功能
+ bar2 单元平行于Z轴时,需要调整局部矢量轴(默认平行Z轴, 故当bar2平行与Z轴时, 无法定义局部bar2自身局部坐标系)

-----------------
### 控制参数
无

-----------------
### 操作
1. 点击计算

-----------------
## 原理
+ 找bar2
+ 根据bar2着node
+ 根据bar2两点node确定矢量
+ 若矢量 X,Y 都接近0, 则方向需调整
