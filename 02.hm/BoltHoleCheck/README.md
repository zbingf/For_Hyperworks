# BoltHoleCheck

### 软件版本
+ hypermesh 2017

### 功能
+ 螺栓孔-孔位对称性检查

-----------------
### 控制参数
+ 允许的角度偏差 deg (面的法向与Bar2的夹角)


-----------------
### 操作
1. 设置控制参数
2. 计算

-----------------
## 原理
+ 测量孔面的法向 与 Bar2的夹角
+ 通过点到距离, 结合面的法向, 生成新的矢量 V1
+ 创建Bar2两点的矢量 V2
+ 获取V1与V2的夹角
